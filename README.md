# Nuxt3 template
> Template for Nuxt3 projects with typescript eslint + prettier and git hooks

## Project details
Template repository for Nuxt3 (Vue3) + Typescript projects

## Project setup
```bash
# Install all dependencies
$ yarn

# To start dev server
$ yarn dev
```

## Project building
```bash
# Build project and all dependencies
$ yarn build

# Run builded project
$ yarn start
```
